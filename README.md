## 本地搭建开发流程

+ 安装docker
+ 安装postman，导入本项目的doc/postman配置.json
+ 安装谷歌浏览器draw.io插件，打开本项目的doc/design.xml 设计文件

+ 配置好mysql
+ 利用bin/minio.sh 运行minio
+ 把minio的配置，手动写入数据库miniocon表

+ 启动本程序
```
go run app.go bin/config.json
```
